Projects

SCALE Student-Centered Adaptive Learning Engine

SCALE is a system for providing a personalized learning experience for students using educational software. This technology can learn from the students themselves and improve over time using data-driven methodologies and a human-centered approach. While it has been long known that personalized educational software is more effective, it is rarely implemented due to the traditionally high cost of content development. SCALE provides machine learning techniques and algorithms, based on award-winning, cutting edge research, that dramatically reduces the cost of development and thereby allows the widespread implementation of truly adaptive computer-based training.

HPIT Hyper Personalized Interactive Tutor

HPIT is a web service that provides developers of educational software the artificial intelligence required to make their products personalize to students. HPIT is a highly distributed environment for facilitating the ability to create new plugins that can connect and interact with the system in a clean a robust manner. TutorGen, in addition to developing the HPIT architecture, is also building plugins that implement the most up-to-date methodologies and algorithms based on the latest research in ITS (Intelligent Tutoring Systems) and EDM (Educational Data Mining) in a cost-effective and widely accessible manner.

LOGIC Linkage Objects for Generalized Instruction in Coding

LOGIC represents a more generalized and data-driven approach towards adaptive instruction and hint feedback for learning programming languages. LOGIC is currently under development for teaching the Java programming language, but the methods being developed are equally applicable to other languages. The key innovation of the LOGIC system is the use of linkage objects that store the most common and effective solutions for programming problems from across thousands of students, in order to provide just-in-time feedback and hints that improve student engagement and success.
